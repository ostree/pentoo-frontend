## Stack
* ReactJs
* SCSS (BEM)
* Material Design
* RamdaJs

## Develop

1. Clone this repo to your local machine
2. `cd pentoo-frontend (or whatever you named it)`
3. Once the packages are done, you can run `npm run start (yarn start)`
4. Go to `localhost:8000`
